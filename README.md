## Help to translate?

### Add lang file
Add your `{lang}.json` to [`public/lang`](/public/lang)

(Replace `{lang}` to the `Locale code`, just like in Minecraft)

Add the file name to [`public/lang/all.json`](/public/lang/all.json)
